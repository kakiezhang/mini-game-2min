# 《120秒下班》MVP 数据配置表 v0.1

## 1. 配置说明

本文件提供 MVP v0.1 的第一版开发数值。所有数值用于首轮可玩原型，不代表最终平衡。

基础单位：

| 项目 | 单位 |
|---|---|
| 时间 | 秒 |
| 坐标 | 像素 |
| 速度 | 像素/秒 |
| 角速度 | 弧度/秒 |
| 血量/伤害 | 数值点 |

坐标系：

```text
原点在地图左上角
x 向右增加
y 向下增加
```

## 18. 升级选项池

| optionId | 技能 | 条件 | 标题 | 说明 |
|---|---|---|---|---|
| keyboardDartUpgrade | keyboardDart | 当前等级 < 5 | 键盘飞镖升级 | 提升键帽数量、伤害或范围 |
| clickerUnlockOrUpgrade | mouseClicker | 当前等级 < 5 | 鼠标连点器 | 自动攻击最近敌人 |
| coffeeDashUnlockOrUpgrade | coffeeDash | 当前等级 < 5 | 咖啡冲刺 | 主动冲刺逃离包围 |
| resignationUnlockOrUpgrade | resignationWave | 当前等级 < 5 | 离职信清屏 | 周期性释放范围冲击波 |

随机规则：

```text
每次升级从可升级选项中随机 3 个
如果可选项不足 3 个，则展示全部
已满级技能不再出现
初始技能键盘飞镖默认 Lv1
其他技能第一次选择时解锁 Lv1
```

## 21. 首轮重点调参项

首轮试玩后优先调整：

| 项目 | 目标 |
|---|---|
| bug.hp | 前 20 秒击杀要轻快 |
| stage1.spawnInterval | 前期敌人密度不能太空 |
| exp level 1 | 第一次升级控制在 15-20 秒 |
| player.baseSpeed | 工位到会议室约 6-8 秒 |
| boss.speed | 老板能制造压力，但不能必死 |
| evacuationHoldTime | 2 秒读条有压力但可完成 |
| maxEnemies | 保证 90 秒后不卡顿 |
