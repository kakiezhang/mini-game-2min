# 《120秒下班》MVP 数据配置表 v0.1

## 1. 配置说明

本文件提供 MVP v0.1 的第一版开发数值。所有数值用于首轮可玩原型，不代表最终平衡。

基础单位：

| 项目 | 单位 |
|---|---|
| 时间 | 秒 |
| 坐标 | 世界单位 |
| 速度 | 世界单位/秒 |
| 角速度 | 弧度/秒 |
| 血量/伤害 | 数值点 |

坐标系：

```text
玩法平面使用 x/z
x 向右增加
z 向下增加
Three.js 世界坐标中 y 表示高度
```

## 18. 升级选项池

| optionId | 技能 | 条件 | 标题 | 说明 |
|---|---|---|---|---|
| firepowerCalibration | defaultSmg | 当前等级 < 5 | 火力校准 | 提升伤害、射速、稳定性或暴击 |
| magazineManagement | defaultSmg | 当前等级 < 5 | 弹匣管理 | 提升弹匣容量和换弹效率 |
| overtimePiercing | defaultSmg | 当前等级 < 5 | 加班穿透弹 | 提升敌人穿透和高价值目标伤害 |
| coffeeDashUnlockOrUpgrade | coffeeDash | 当前等级 < 5 | 咖啡冲刺 | 主动冲刺逃离包围 |

随机规则：

```text
每次升级从可升级选项中随机 3 个
如果可选项不足 3 个，则展示全部
已满级技能不再出现
默认冲锋枪始终存在，三条枪械强化初始均为 Lv0
咖啡冲刺作为初始主动技能 Lv1
automatic 模式保留，但当前升级池不提供具体自动攻击装备
```

## 21. 首轮重点调参项

首轮试玩后优先调整：

| 项目 | 目标 |
|---|---|
| bug.hp | 前 20 秒击杀要轻快 |
| stage1.spawnInterval | 前期敌人密度不能太空 |
| defaultSmg.damage | 普通怪击杀所需子弹数可理解且稳定 |
| defaultSmg.reloadTime | 换弹有压力但不会频繁打断操作 |
| ammoBoxRespawnInterval | 固定补给足以避免永久断粮，仍保留路线选择 |
| exp level 1 | 第一次升级控制在 15-20 秒 |
| player.baseSpeed | 工位到会议室约 6-8 秒 |
| boss.speed | 老板能制造压力，但不能必死 |
| evacuationHoldTime | 2 秒读条有压力但可完成 |
| maxEnemies | 约 30 只时门洞仍可通行，手动瞄准仍有空间 |
