# 《120秒下班》MVP 需求文档 v0.1

## 1. 项目概述

《120秒下班》是一款竖屏低模 3D 斜俯视办公室题材割草撤离小游戏。

游戏表现层采用低模 3D 场景、固定斜俯视相机和办公室箱庭地图；玩法逻辑仍然按平面移动处理，玩家和敌人在地面平面上移动、追击、攻击和撤离。

玩家扮演一名打工人，在 120 秒倒计时内，于办公室中对抗不断出现的 Bug、需求变更、会议怪和老板，寻找门禁卡，等待电梯开放，并在强制加班前成功进入电梯下班。

游戏主打短局、解压、爽感和轻度逃离压力。

## 2. 游戏定位

| 项目 | 内容 |
|---|---|
| 游戏名 | 《120秒下班》 |
| 类型 | 低模 3D 斜俯视割草撤离小游戏 |
| 主题 | 打工人、办公室、准时下班、解压 |
| 核心体验 | 清怪、升级、拿门禁卡、冲电梯、躲老板 |
| 单局时长 | 120 秒 |
| 屏幕方向 | 竖屏 |
| 画面表现 | 低模 3D、斜俯视相机、办公室箱庭 |
| 攻击方式 | 自动攻击 |
| 玩家操作 | 控制移动为主 |
| 风格 | 轻松、搞笑、解压、节奏快、低模办公室荒诞感 |

### 2.1 表现形式补充

MVP 的目标画面不是平面 2D 色块地图，而是类似俯视战术/撤离游戏的低模 3D 地图形式：

```text
3D 地面
3D 墙体
3D 办公桌、会议桌、咖啡机、电梯
固定斜俯视相机
真实高度差、基础光照和阴影
```

但 MVP 不引入复杂模型制作流程，第一版优先使用 Three.js 基础几何体搭建低模场景。

## 4. 核心玩法

玩家在办公室地图中移动，角色自动攻击附近敌人。

敌人会持续刷新并追击玩家。玩家击杀敌人获得经验，升级后选择技能强化。

在游戏第 35 秒，门禁卡刷新在会议室。玩家需要前往会议室拾取门禁卡。

在游戏第 80 秒，电梯开放。玩家持有门禁卡后，需要前往电梯区域并站立读条 2 秒完成撤离。

第 90 秒，老板从老板办公室出现并追击玩家。

如果玩家在 120 秒前成功进入电梯，则下班成功。  
如果玩家死亡，或者 120 秒结束仍未撤离，则失败，被迫加班。

## 6. 操作方式

### 6.1 手机端操作

手机端使用左下角虚拟摇杆控制玩家移动。

普通攻击自动释放，玩家不需要手动瞄准和攻击。

右下角预留一个技能按钮，用于释放“咖啡冲刺”。

| 操作 | 效果 |
|---|---|
| 拖动虚拟摇杆 | 控制玩家移动方向 |
| 松开虚拟摇杆 | 玩家停止移动 |
| 点击右下角按钮 | 释放咖啡冲刺 |

### 6.2 PC Web 操作

PC Web 端支持键盘移动。

| 按键 | 效果 |
|---|---|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| Space | 释放咖啡冲刺 |

### 6.3 输入系统原则

手机虚拟摇杆和 PC 键盘都统一输出一个平面移动方向：

```text
moveDirection = (x, z)
```

玩家移动系统只处理 `moveDirection`，不区分输入来源。Three.js 世界坐标中，`x/z` 表示地面平面，`y` 表示高度。

## 7. 地图设计

### 7.1 地图定位

第一版地图为一张竖屏适配的小型低模 3D 办公室箱庭地图。

地图不是长条形，而是保留办公室房间布局，有横向空间和路线选择。

地图表现采用 3D 盒体和低模物件构建，玩家从固定斜俯视相机观察办公室空间。

### 7.2 地图结构

```text
┌───────────────┬───────────────┐
│    会议室      │   老板办公室   │
│  门禁卡刷新点  │   老板出生点   │
├───────────────┼───────────────┤
│    工位区      │    茶水间      │
│    玩家出生点  │    咖啡补给点  │
├───────────────┴───────────────┤
│            电梯间              │
│            撤离点              │
└───────────────────────────────┘
```

### 7.3 区域功能

| 区域 | 位置 | 功能 |
|---|---|---|
| 工位区 | 左中 | 玩家出生点，基础战斗区 |
| 会议室 | 左上 | 门禁卡刷新点，会议怪主要区域 |
| 老板办公室 | 右上 | 老板出生点，高风险区域 |
| 茶水间 | 右中 | 咖啡补给点 |
| 电梯间 | 下方 | 撤离点 |

### 7.4 地图尺寸建议

| 项目 | 建议 |
|---|---|
| 地图宽度 | 约 1.2～1.5 个屏幕宽，映射到 3D 世界 x 轴 |
| 地图高度 | 约 1.5～1.8 个屏幕高，映射到 3D 世界 z 轴 |
| 工位到会议室 | 6～8 秒 |
| 工位到茶水间 | 5～7 秒 |
| 工位到电梯 | 6～8 秒 |
| 会议室到电梯 | 10～12 秒 |
| 绕完整图 | 20～30 秒 |

### 7.5 障碍物设计

第一版只放少量 3D 障碍物，避免手机操作卡手。

| 区域 | 障碍物 |
|---|---|
| 工位区 | 低模办公桌、椅子 |
| 会议室 | 低模会议桌 |
| 茶水间 | 咖啡机模型/盒体 |
| 老板办公室 | 老板桌、文件柜 |
| 电梯间 | 电梯门、等待区域、灯箱 |

## 10. 技能设计

第一版做 4 个技能。

### 10.1 键盘飞镖

初始技能，近身环绕攻击。

| 等级 | 效果 |

| Lv1 | 2 个键帽围绕玩家旋转 |
| Lv2 | 键帽数量 +1 |
| Lv3 | 伤害提升 |
| Lv4 | 旋转范围提升 |

### 10.2 鼠标连点器

远程自动攻击技能。

| 等级 | 效果 |
|---|---|
| Lv1 | 每 1 秒向最近敌人发射 1 个鼠标指针 |
| Lv2 | 射速提升 |
| Lv3 | 子弹穿透 +1 |
| Lv4 | 同时发射 2 个鼠标指针 |
| Lv5 | 伤害提升 |

### 10.3 咖啡冲刺

主动逃生技能。

| 等级 | 效果 |
|---|---|
| Lv1 | 向当前移动方向冲刺一段距离 |
| Lv2 | 冷却缩短 |

| Lv4 | 冲刺后短暂提升移动速度 |
| Lv5 | 可连续冲刺 2 次 |

### 10.4 离职信清屏

大范围解压技能。

| 等级 | 效果 |
|---|---|
| Lv1 | 每 25 秒释放一次离职信冲击波 |
| Lv2 | 冷却缩短 |

| Lv4 | 伤害提升 |

## 12. 升级系统

玩家击杀敌人获得经验。

经验条满后升级，游戏暂停，弹出 3 个随机技能强化选项。

玩家选择 1 个后继续游戏。

| 项目 | 设计 |
|---|---|
| 一局升级次数 | 5～8 次 |
| 第一次升级 | 15～20 秒内 |
| 升级选项 | 3 选 1 |
| 技能池 | 4 个技能的升级项 |
| 升级时是否暂停 | 暂停 |
| 选择后 | 立即生效 |

## 13. 撤离系统

### 13.1 电梯状态

| 时间 | 状态 |
|---|---|
| 0～80 秒 | 电梯关闭 |
| 80 秒后 | 电梯开放 |
| 玩家无门禁卡 | 无法撤离 |
| 玩家有门禁卡 | 可进入电梯读条撤离 |

### 13.2 撤离规则

玩家持有门禁卡并进入电梯区域后，显示撤离读条。

读条时间：2 秒。

读条完成后，进入成功结算。

读条期间玩家仍可能受到伤害。

## 16. 成功 / 失败规则

### 16.1 成功条件

同时满足：

1. 玩家持有门禁卡。
2. 电梯已开放。
3. 玩家进入电梯区域并完成 2 秒读条。
4. 玩家未死亡。

成功后显示：

> 下班成功！今日无事发生。

### 16.2 失败条件

任意满足：

1. 玩家血量归零。
2. 120 秒倒计时结束仍未成功撤离。

失败文案：

> 你被迫加班了。

或：

> 你被工作压垮了。

## 19. MVP 验收标准

第一版做完后，至少要满足这些标准：

| 验收项 | 标准 |
|---|---|
| 可玩闭环 | 能从开始进入一局，成功或失败结算 |
| 操作可用 | 手机虚拟摇杆和 PC 键盘都能控制移动 |
| 节奏明确 | 120 秒内有清怪、拿卡、冲电梯、老板追击 |
| 目标清楚 | 玩家知道门禁卡在哪、电梯在哪 |
| 爽感初步成立 | 前 20 秒能快速击杀敌人并升级 |
| 压力成立 | 90 秒老板出现后有逃离紧张感 |
| 撤离成立 | 进入电梯并读条后能成功下班 |
| 失败成立 | 死亡或超时能进入失败结算 |
| 移动端可玩 | 竖屏下 UI 不遮挡核心操作 |
| 3D 可读性 | 斜俯视相机下，玩家、敌人、门禁卡、电梯和障碍物清晰可辨 |
| PC 可调试 | Web 端键盘操作正常 |
