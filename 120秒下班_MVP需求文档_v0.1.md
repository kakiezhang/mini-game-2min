# 《120秒下班》MVP 需求文档 v0.1

> 2026-06-20 战斗方向调整：默认主武器改为手动枪械射击，并加入有限弹药与补给管理；战斗系统继续保留自动攻击能力，供后续装备使用。

## 1. 项目概述

《120秒下班》是一款竖屏低模 3D 斜俯视办公室题材手动射击撤离小游戏。

游戏表现层采用低模 3D 场景、固定斜俯视相机和办公室箱庭地图；玩法逻辑仍然按平面移动处理，玩家和敌人在地面平面上移动、追击、攻击和撤离。

玩家扮演一名打工人，在 120 秒倒计时内，于办公室中对抗不断出现的 Bug、需求变更、会议怪和老板，寻找门禁卡，等待电梯开放，并在强制加班前成功进入电梯下班。

游戏主打短局、手动射击、弹药管理、解压爽感和轻度逃离压力。

### 2.1 表现形式补充

MVP 的目标画面不是平面 2D 色块地图，而是类似俯视战术/撤离游戏的低模 3D 地图形式：

```text
3D 地面
3D 墙体
3D 办公桌、会议桌、咖啡机、电梯
固定斜俯视相机
真实高度差、基础光照和阴影
```

但 MVP 不引入复杂模型制作流程，第一版优先使用 Three.js 基础几何体搭建低模场景。

## 4. 核心玩法

玩家在办公室地图中移动，使用默认枪械手动瞄准敌人并主动开火。默认枪械不自动索敌或自动开火，但战斗系统需要保留自动攻击触发模式，供后续副装备或技能扩展。

枪械使用有限弹匣和后备弹药，射空后必须换弹，玩家需要通过地图固定补给和敌人掉落补充子弹。

敌人会持续刷新并追击玩家。玩家击杀敌人获得经验，升级后选择枪械或逃生能力强化。

在游戏第 35 秒，门禁卡刷新在会议室。玩家需要前往会议室拾取门禁卡。

在游戏第 80 秒，电梯开放。玩家持有门禁卡后，需要前往电梯区域并站立读条 2 秒完成撤离。

第 90 秒，老板从老板办公室出现并追击玩家。

如果玩家在 120 秒前成功进入电梯，则下班成功。  
如果玩家死亡，或者 120 秒结束仍未撤离，则失败，被迫加班。

## 6. 操作方式

### 6.1 手机端操作

手机端使用左下角虚拟摇杆控制移动和朝向，右下角独立射击按钮控制开火。

玩家移动时，枪口自动朝向当前移动方向；停止移动后保留最后有效朝向。按住射击按钮时连续开火，松开后立即停火。

咖啡冲刺和换弹按钮放在射击按钮上方，避免与移动摇杆重叠。

| 操作 | 效果 |
|---|---|
| 拖动左摇杆 | 控制玩家移动方向，同时改变枪口朝向 |
| 松开左摇杆 | 玩家停止移动，保留最后朝向 |
| 按住射击按钮 | 朝当前方向连续射击 |
| 松开射击按钮 | 停止射击 |
| 点击换弹按钮 | 主动更换弹匣 |
| 点击技能按钮 | 释放咖啡冲刺 |

### 6.2 PC Web 操作

PC Web 端默认使用键盘移动并决定朝向，适合无鼠标的笔记本电脑。触控板或鼠标移动可在角色停止时临时调整朝向，作为可选增强输入。

| 按键 | 效果 |
|---|---|
| W / ↑ | 向上移动 |
| S / ↓ | 向下移动 |
| A / ← | 向左移动 |
| D / → | 向右移动 |
| J / 鼠标左键 | 射击 |
| 移动触控板或鼠标 | 玩家停止移动时可调整枪口方向 |
| R | 主动换弹 |
| Space | 释放咖啡冲刺 |

### 6.3 输入系统原则

手机与 PC 输入统一输出以下战斗指令：

```text
moveDirection = (x, z)
aimDirection = (x, z)
firePressed = true / false
reloadPressed = true / false
dashPressed = true / false
```

`aimDirection` 默认取最后一个有效的 `moveDirection`；只有角色停止移动且收到指针移动时，才使用指针与地面的交点更新。移动系统、武器系统和技能系统只消费统一指令，不区分输入来源。

## 7. 地图设计

地图表现采用 3D 盒体和低模物件构建，玩家从固定斜俯视相机观察办公室空间。

### 9.5 敌人密度原则

手动射击需要留出瞄准、换弹和穿越门洞的空间。MVP 场上同时存活敌人建议控制在 25～35 只，不再采用百只敌人围堵玩家的自动割草密度。

敌人数量减少后，通过敌人类型、移动速度、路线封锁和老板追击提供压力。普通敌人刷新点不能与玩家距离过近，也不能落在墙体、家具或门洞通行区域内。

## 10. 枪械与技能设计

第一版只提供 1 把默认枪械，不做武器切换、武器背包和配件安装。升级池包含 3 条枪械强化路线和 1 条主动逃生路线，枪械升级不得把默认枪械变成自动索敌武器。

战斗系统需支持两种攻击触发模式：

| 模式 | 触发方式 | MVP 使用范围 |
|---|---|---|
| manual | 玩家提供瞄准方向并主动开火 | 默认冲锋枪 |
| automatic | 系统按装备规则自动选取目标或攻击 | 保留能力，暂不配置具体自动攻击装备 |

自动攻击是装备级能力，不代表玩家持有的所有武器都会自动攻击。飞镖暂时移出 MVP 装备清单，后续确定副装备体系时再单独设计。

### 10.1 默认冲锋枪

默认武器使用射线命中判定，并显示枪口火焰、曳光线、命中粒子、轻微后坐力和镜头反馈。射线会被墙体和不可穿透家具阻挡。

| 项目 | 初始值建议 |
|---|---:|
| 弹匣容量 | 20 发 |
| 初始后备弹药 | 80 发 |
| 后备弹药上限 | 120 发 |
| 单发伤害 | 18 |
| 射速 | 每秒 5 发 |
| 换弹时间 | 1.3 秒 |
| 射击方式 | 玩家瞄准后按住射击，可连续开火 |
| 空仓行为 | 停止射击并提示换弹，不生成免费子弹 |

玩家可随时主动换弹。弹匣射空后允许自动开始换弹，但换弹不会补充后备弹药之外的子弹。

换弹期间允许移动，但不能射击；使用咖啡冲刺会取消当前换弹。战术换弹保留弹匣中的剩余子弹，只从后备弹药补足缺口。

### 10.2 火力校准

| 等级 | 效果 |
|---|---|
| Lv1 | 单发伤害提升 |
| Lv2 | 射速小幅提升 |
| Lv3 | 单发伤害再次提升 |
| Lv4 | 降低连续射击散布 |
| Lv5 | 暴击概率提升 |

### 10.3 弹匣管理

| 等级 | 效果 |
|---|---|
| Lv1 | 弹匣容量提升 |
| Lv2 | 换弹时间缩短 |
| Lv3 | 弹匣容量再次提升 |
| Lv4 | 换弹后短暂提升射速 |
| Lv5 | 空仓换弹速度额外提升 |

升级弹匣容量不凭空增加后备弹药，只改变单个弹匣可装填数量。

### 10.4 加班穿透弹

| 等级 | 效果 |
|---|---|
| Lv1 | 子弹可额外穿透 1 个普通敌人 |
| Lv2 | 穿透后的伤害衰减降低 |
| Lv3 | 对精英和老板伤害提升 |
| Lv4 | 子弹可额外穿透 2 个普通敌人 |
| Lv5 | 命中时小范围击退普通敌人 |

墙体、会议桌等硬障碍始终阻挡子弹，穿透升级只作用于敌人。

### 10.5 咖啡冲刺

主动逃生技能。

| 等级 | 效果 |
|---|---|
| Lv1 | 向当前移动方向冲刺一段距离 |
| Lv2 | 冷却缩短 |
| Lv3 | 冲刺时撞开敌人 |
| Lv4 | 冲刺后短暂提升移动速度 |
| Lv5 | 可连续冲刺 2 次 |

第一版建议手机在射击按钮上方设置独立按钮，PC Web 使用 Space 释放。

必要战斗补给。

| 项目 | 设计 |
|---|---|

## 12. 升级系统

玩家击杀敌人获得经验。

经验条满后升级，游戏暂停，弹出 3 个随机技能强化选项。

玩家选择 1 个后继续游戏。

| 项目 | 设计 |
|---|---|
| 一局升级次数 | 5～8 次 |
| 第一次升级 | 15～20 秒内 |
| 升级选项 | 3 选 1 |
| 技能池 | 3 条枪械强化路线 + 咖啡冲刺 |
| 升级时是否暂停 | 暂停 |
| 选择后 | 立即生效 |

## 13. 撤离系统

### 13.1 电梯状态

| 时间 | 状态 |
|---|---|
| 0～80 秒 | 电梯关闭 |
| 80 秒后 | 电梯开放 |
| 玩家无门禁卡 | 无法撤离 |
| 玩家有门禁卡 | 可进入电梯读条撤离 |

### 13.2 撤离规则

玩家持有门禁卡并进入电梯区域后，显示撤离读条。

读条时间：2 秒。

读条完成后，进入成功结算。

读条期间玩家仍可能受到伤害。

## 16. 成功 / 失败规则

### 16.1 成功条件

同时满足：

1. 玩家持有门禁卡。
2. 电梯已开放。
3. 玩家进入电梯区域并完成 2 秒读条。
4. 玩家未死亡。

成功后显示：

> 下班成功！今日无事发生。

### 16.2 失败条件

任意满足：

1. 玩家血量归零。
2. 120 秒倒计时结束仍未成功撤离。

失败文案：

> 你被迫加班了。

或：

> 你被工作压垮了。

## 19. MVP 验收标准

第一版做完后，至少要满足这些标准：

| 验收项 | 标准 |
|---|---|
| 可玩闭环 | 能从开始进入一局，成功或失败结算 |
| 操作可用 | 手机单摇杆和 PC 纯键盘都能完成移动、转向、射击与换弹 |
| 节奏明确 | 120 秒内有清怪、拿卡、冲电梯、老板追击 |
| 目标清楚 | 玩家知道门禁卡在哪、电梯在哪 |
| 射击成立 | 枪口方向、准星和命中位置一致，墙体能够阻挡射击 |
| 手动枪械成立 | 默认枪械不会自动索敌或自动开火，停止射击输入后枪械立即停火 |
| 自动攻击可扩展 | 战斗接口保留 manual / automatic 触发模式，当前无需配置具体自动装备 |
| 弹药成立 | 每次射击消耗子弹，空仓不能开火，换弹只消耗后备弹药 |
| 补给成立 | 固定弹药点能避免玩家因随机掉落失败而永久断粮 |
| 爽感初步成立 | 前 20 秒手动射击反馈清晰，能击杀敌人并升级 |
| 压力成立 | 90 秒老板出现后有逃离紧张感 |
| 撤离成立 | 进入电梯并读条后能成功下班 |
| 失败成立 | 死亡或超时能进入失败结算 |
| 移动端可玩 | 竖屏下移动摇杆、射击、换弹和技能按钮不重叠、不遮挡核心视野 |
| 3D 可读性 | 斜俯视相机下，玩家、敌人、门禁卡、电梯和障碍物清晰可辨 |
| PC 可调试 | Web 端纯键盘可移动、转向、射击和换弹，指针瞄准作为可选增强正常 |
